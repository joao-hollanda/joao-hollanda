### 🚀 Projeto Principal — Learnly

</div>

> **Learnly** é uma plataforma de preparação para o ENEM com foco em simulados adaptativos e feedback individual gerado por IA. Desenvolvida do zero com arquitetura profissional e visão de produto real.

<table>
<tr>
<td width="50%">

**🏗️ Arquitetura**
- Clean Architecture (camadas bem definidas)
- Use Cases / Repositories / Controllers desacoplados
- JWT com HttpOnly Cookies (segurança first)
- IDOR prevention via JWT claims

</td>
<td width="50%">

**🤖 IA Integrada**
- Feedback por questão via Groq API
- `llama-3.1-8b-instant` para baixa latência
- Chamada única em batch por simulado
- Análise de desempenho por área de conhecimento

</td>
</tr>
<tr>
<td width="50%">

**⚡ Performance**
- TanStack Query com `staleTime: Infinity`
- `invalidateQueries` cross-page coordenado
- PostgreSQL/Neon com queries otimizadas
- Paginação e N+1 prevention via EF Core

</td>
<td width="50%">

**📈 Tração**
- Apresentação acadêmica → oferta de escola real
- Análise de load testing com k6 / Artillery
- Roadmap de escalabilidade em andamento
- Pitch page com market sizing e modelo de negócio

</td>
</tr>
</table>
